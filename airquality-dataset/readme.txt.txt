Description of the fields:
 - created_at : Timestamp for the observation in UTC
 - pm2_5 contains PM2.5 concentrations(µg/m3) for the respective timestamp from sensor one
-  pm10 contains PM10 concentrations(µg/m3) for the respective timestamp from sensor one
-  s2_pm2_5 contains PM2.5 concentrations(µg/m3) for the respective timestamp from sensor two	
-  s2_pm10 contains PM2.5 concentrations(µg/m3) for the respective timestamp from sensor two
